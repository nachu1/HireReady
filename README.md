# HireReady – AI Resume Analyzer & Interview Coach

## Overview
HireReady is an AI-powered web application that helps job seekers analyze resumes, improve them, generate ATS-friendly resumes, create cover letters, prepare interview questions, and interact with an AI career assistant.

## Features
- User Registration & Login
- Resume Upload (PDF)
- AI Resume Analysis
- ATS Feedback
- Resume Improvement
- AI Resume Generation
- Cover Letter Generator
- Mock Interview Questions
- AI Career Chat
- Resume PDF Download

## Tech Stack
Frontend
- React
- Vite
- Tailwind CSS
- Axios

Backend
- FastAPI
- Python
- MongoDB
